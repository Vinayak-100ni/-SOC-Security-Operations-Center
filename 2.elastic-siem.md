## 1. What is SIEM?

**SIEM = Security Information and Event Management**

SIEM is a system that:

* Collects logs from many devices
* Stores them in one place
* Finds threats or suspicious activity
* Alerts security teams

**In short:**

> SIEM = Log collection + Detection + Alerting + Investigation
